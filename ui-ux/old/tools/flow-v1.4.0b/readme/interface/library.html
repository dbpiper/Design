Library¶

Library is the place where Default Curves or your Custom Presets live in. It’s one of the most frequently used parts of Flow.

Double-click library curves to quickly apply them! Use keyboard shortcuts to manage ease direction:

SHIFT Applies to Ease In
ALT Applies to Ease Out
CTRL Applies to Easy Ease

Drag & Drop items in the Library to rearagne them to your needs.

Change the Library Item Size in Preferences, or with +- buttons in the interface.

Default Curves¶

25 pre-defined, common curves for every day use

User Curves¶

All of your custom or imported curves live here

Clear Library¶

Removes all custom curves from the user library

Export Library¶

Exports your custom Flow library

Import Library¶

Imports an exported Flow library

Tip

Check out extensive list of great curves on Flow’s product page https://aescripts.com/flow/​

Context Menu¶

Right click in the library to access additional options to:

Edit Item Edit Bezier values manually
Rename Item Change library item’s name
Remove Item Remove library item from the list
Sort Library Sort all items in the library by Date or alphabetically in Ascending or Descending order
Manage Library Import Library, Export Library, Clear Library
