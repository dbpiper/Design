Appearance¶

Flow Theme¶

Use this to change Flow colour theme!

Flow Flow’s native theme
AE AE-based theme; adapts to AE brightness!

Show Last Curve in Graph¶

When enabled the graph editor displays previously used curve.

Show Tooltips¶

Toggles tooltips visibility on or off in Flow’s interface.

Show Library IO Buttons¶

Toggles visibility “on” and “off” for Clear, Export, Import Library buttons in User Curves library.

Show Library Item Resize Buttons¶

Toggles visibility “on” and “off” for +- buttons in the Library.

Library Item Size¶

Changes the thumbnail size in the library.

Range between 20 to 50 pixels.
